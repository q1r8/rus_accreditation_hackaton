# Запуск сервиса
## Необходим python3.8 и Node.js LTS

Чтобы загрузить модели и необходимые файлы выполните команду:
sh load_model.sh

*   python3 -m virtualenv venv
*   source venv/bin/activate
*   pip install -r requirements.txt

*   cd frontend
*   npm i

# Для запуска фронта
*   npm start

# Для запуска бекенда
*   python app.py
